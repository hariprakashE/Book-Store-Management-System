Book Store Management System – HTML, Java, Spring Boot, MySQL

•	Developed a web-based Book Store Management System using Java, Spring Boot, and MySQL. 

•	Implemented CRUD operations to add, update, delete, and manage book records efficiently. 

•	Designed a responsive user interface using HTML, CSS, Bootstrap, and Thymeleaf. 

•	Followed MVC architecture and integrated MySQL database using Spring Data JPA for secure data management.
